32 layers total: 24 DeltaNet (linear O(1) in seq) + 8 Gated Attention (quadratic O(seq²))

Why this matters for attention memory

Gated Attention score matrix: batch × Q_heads × seq × seq × 2 bytes
Gated DeltaNet state matrix: batch × QK_heads × head_dim × head_dim × 2 bytes [O(1) in seq!]
At seq=262144: Attention score per layer = batch × 16 × 262144² × 2 ≈ massive
DeltaNet state per layer = batch × 16 × 128 × 128 × 2 = 0.53 MB [constant!]

Pre-training vs SFT: what you're actually training on

Pre-training (next-token prediction)

Input: [The cat sat on the mat]
Target: [cat sat on the mat .]
Loss: every token position
No concept of "prompt" or "response"
Documents packed end-to-end

SFT (instruction fine-tuning)

Input: [INST] prompt [/INST]
Target: desired_response
Loss: response tokens only (masked)
Prompt tokens consume memory
but don't produce gradients
